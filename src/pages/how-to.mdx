# How-to

This section describes various real-life situations and issues that you may
experience with Unit.

- [Unit in Docker](howto/docker): Configure a standalone Unit or a Unit-run app in a Docker
  container.
- [Building From Source](howto/source): Build Unit and its language modules from source code.
- [Unit in Ansible](howto/ansible): Use a third-party Ansible collection to automate Unit
  deployment.
- [NGINX Integration](howto/integration): Front or secure Unit with NGINX.
- [TLS with Certbot](howto/certbot): Use EFFâ€™s Certbot with Unit to simplify certificate
  manipulation.
- [Working With Language Modules](howto/modules): Build new modules or prepare custom packages for
  Unit.
- [App Samples](howto/samples): Reuse sample app configurations for all languages
  supported by Unit.
- [Security Checklist](howto/security): Recommendations and considerations for hardening Unit.
- [Walkthrough](howto/walkthrough): Follow an end-to-end guide to application configuration
  in Unit.
</div>
